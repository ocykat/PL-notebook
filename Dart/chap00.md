# Chapter 0: Introduction

## Overview
* Object-Oriented
* Statically-Typed
* Runtime Environments:
  * **Web app**: Dart code => JS code => Run in browser
  * **PC**:      Dart code => Dart Virtual Machine: Dart code is executed in a command-line interface environment
